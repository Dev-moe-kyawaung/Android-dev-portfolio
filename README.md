# Android Developer Portfolio

A high-conversion Flutter portfolio showcasing 5 top Android projects with technical stack, design challenges, and performance metrics.

## Features

- ✅ Hero section with value proposition
- ✅ 5 featured projects with metrics
- ✅ Screenshot carousel per project
- ✅ GitHub + Play Store buttons
- ✅ Interactive tech stack tag-cloud
- ✅ Responsive mobile/tablet/desktop
- ✅ Dark Mode + Material 3

## Setup

1. Install Flutter 3.0+
2. Clone repo:
   ```bash
   git clone https://github.com/Dev-Moe-kyawaung/Android-dev-portfolio
   cd portfolio

   Install dependencies:
   flutter pub get

   Run for Flutter Web:
flutter run -d chrome

Deploy to GitHub Pages:

flutter build web
git add .
git commit -m "Deploy portfolio"
git push
